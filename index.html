Autenticación
Regístrate o inicia sesión. Una vez autenticado, el token JWT se guarda automáticamente y se envía en cada solicitud siguiente en el header Authorization: Bearer <token>.
Todo usuario se registra como CLIENTE por defecto. Un ADMIN puede promover usuarios desde la sección Administración.
